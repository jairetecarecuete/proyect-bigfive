# Proyecto Big Five 🧠

Test de personalidad basado en el modelo **Big Five (OCEAN)**, desarrollado en Python como proyecto final de mi curso de programación.

## ¿Qué hace?

El programa aplica un cuestionario de personalidad basado en las 5 grandes dimensiones psicológicas:

- **O**penness (Apertura)
- **C**onscientiousness (Responsabilidad)
- **E**xtraversion (Extraversión)
- **A**greeableness (Amabilidad)
- **N**euroticism (Neuroticismo)

A partir de las respuestas del usuario, calcula el promedio de cada dimensión, guarda los resultados, y genera un reporte visual con gráficos.

## Estructura del proyecto

| Archivo | Función |
|---|---|
| `main.py` | Punto de entrada del programa |
| `modelos.py` | Clases que representan al participante y sus respuestas |
| `preguntas.py` | Banco de preguntas del test |
| `analisis.py` | Cálculo de promedios y generación de gráficos/reportes |

## Tecnologías usadas

- **Python** (Programación Orientada a Objetos)
- **CSV** para guardar los resultados
- **Matplotlib** para generar gráficos
- **HTML** para los reportes visuales

## Cómo ejecutarlo

```bash
python main.py
```
